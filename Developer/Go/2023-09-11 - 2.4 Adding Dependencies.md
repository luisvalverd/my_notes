# 2023-09-11 - 2.4 Adding Dependencies

To add all dependencies for a package of a projects

```bash
go get .
```

to add a specific dependency use

```bash
go get example.com/theirmodule
```
## Example:

```bash
go get github.com/google/uuid
```
